# Käyttöohje

## Asennus ja käynnistys

- Asenna riippuvuudet komennolla ``` poetry install ```
- Käynnistä ohjelma komennolla ``` poetry run invoke start ```
- Odota että Flask-serveri on pystyssä, ja mene sitten osoitteeseen ``` localhost:5000 ```

## Testit
- Aja testit komennolla ```poetry run invoke test ```
- Testikattavuusraportin näet komennolla ``` poetry run invoke coverage ```
