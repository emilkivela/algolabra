## Käyttöohjeet
1. Asenna riippuvuudet komennolla ``` poetry install ```
2. Käynnistä ohjelma komennolla ``` poetry run invoke start ```
3. Odota että Flask-serveri on pystyssä, ja mene sitten osoitteeseen ``` localhost:5000 ```
